Podržani formati elektronskih dokumenata

Elektronski dokumenti su najčešće korišćeni računarski fajlovi. Zahvaljujući veoma razvijenoj modernoj računarskoj mreži, pogodnije je distribuirati elektronske dokumente nego štampane. Danas se koristi mnogo uređaja za prikaz dokumenata, pa postoji mnogo vlasničkih i otvorenih formata fajlova. Document Editor podržava najpopularnije među njima.

Prilikom otpremanja ili otvaranja fajla za uređivanje, on će biti konvertovan u Office Open XML (DOCX) format. Ovo se radi kako bi se ubrzala obrada fajla i povećala interoperabilnost.

Sledeća tabela sadrži formate koji se mogu otvoriti za pregled i/ili uređivanje.

DjVu Format fajla dizajniran prvenstveno za čuvanje skeniranih dokumenata, posebno onih koji sadrže kombinaciju teksta, linijskih crteža i fotografija +
DOC Ekstenzija fajla za dokumente za obradu teksta kreirane u programu Microsoft Word + +
DOCM Microsoft Word dokument sa omogućenim makroima
Ekstenzija fajla za dokumente generisane u Microsoft Word 2007 ili novijim verzijama sa mogućnošću pokretanja makroa		 + +
DOCX Office Open XML
Kompresovani, XML-bazirani format fajla koji je razvio Microsoft za predstavljanje tabela, grafikona, prezentacija i tekstualnih dokumenata		 + +
DOTX Word Open XML šablon dokumenta
Kompresovani, XML-bazirani format fajla koji je razvio Microsoft za šablone tekstualnih dokumenata. DOTX šablon sadrži podešavanja formatiranja, stilove itd. i može se koristiti za kreiranje više dokumenata sa istim formatiranjem		 + +
EPUB Electronic Publication
Besplatan i otvoren standard za e-knjige koji je kreirao International Digital Publishing Forum		 + +
FODT Tekstualni dokument sačuvan u OpenDocument formatu, ali sačuvan kao ravni XML fajl + +
FB2 E-book ekstenzija koja omogućava čitanje knjiga na računaru ili mobilnim uređajima + +
HTML HyperText Markup Language
Glavni markup jezik za web stranice		 + +
HWP Hangul Word Processor
Hanword dokument fajl		 + +
HWPX Hangul Word Processor 2010 dokument
Hanword dokument fajl		 + +
MD MD
Ekstenzija fajla zasnovana na laganom markup jeziku koji se koristi za kreiranje dokumenata u običnom tekstu.		 + +
MHTML MIME (Multipurpose Internet Mail Extensions) HTML
Format web arhive koji objedinjuje sav sadržaj jedne web stranice.		 + +
ODT Format fajla za obradu teksta OpenDocument standarda, otvorenog standarda za elektronske dokumente + +
OFORM Zastareo i trenutno nekorišćen format za popunjavanje formulara. Bićete zamoljeni da ga sačuvate u PDF kako biste mogli da ga popunite. +
OTT OpenDocument šablon dokumenta
OpenDocument format fajla za šablone tekstualnih dokumenata. OTT šablon sadrži podešavanja formatiranja, stilove itd. i može se koristiti za kreiranje više dokumenata sa istim formatiranjem		 + +
PDF Portable Document Format
Format fajla koji se koristi za predstavljanje dokumenata bez obzira na korišćeni softver, hardver i operativni sistem		 +
PDF/A Portable Document Format / A
ISO standardizovana verzija Portable Document Format (PDF) formata specijalizovana za arhiviranje i dugoročno čuvanje elektronskih dokumenata. 		+
RTF Rich Text Format
Format fajla za dokumente koji je razvio Microsoft za razmenu dokumenata između različitih platformi		 + +
STW StarOffice šablon dokumenta
Format fajla koji se koristi za čuvanje dokumenata za obradu teksta u OpenOffice.org Writer-u. STW fajl može se uređivati pomoću bilo kog softvera koji podržava OpenDocument Format (ODF) standard. STW fajlovi sadrže tekst, slike i druge informacije o formatiranju.		 + +
SXW StarOffice/OpenOffice Writer dokument
Format tekstualnog dokumenta uključen u StarOffice paket koji je zasnovan na OpenDocument XML formatu i kompatibilan je sa Apache OpenOffice (ranije OpenOffice.org) Writer-om, naslednikom StarOffice Writer-a.		 + +
TXT Ekstenzija fajla za tekstualne dokumente koji obično sadrže vrlo malo formatiranja + +
WPS Microsoft Works Word Processor tekstualni dokument
Tekstualni dokument kreiran u Microsoft Works Word Processor programu, programu za obradu teksta koji je kasnije zamenjen Microsoft Word-om. WPS fajlovi su slični Word dokumentima, ali imaju manje opcija za formatiranje.		 + +
WPT WordPerfect tekstualni dokument
Tekstualni fajl sa .wpt ekstenzijom sačuvan je u binarnom formatu i može sadržati tekst, grafikone, slike, tabele i druge tekstualne podatke.		 + +
XML Extensible Markup Language (XML).
Jednostavan i fleksibilan markup jezik izveden iz SGML-a (ISO 8879) koji je dizajniran za čuvanje i prenos podataka.		 + +
XPS Open XML Paper Specification
Otvoreni dokument format sa fiksnim rasporedom bez naknade za licencu koji je razvio Microsoft		 +
