Otpremite dokument na WordPress

Možete pisati svoje članke u ONLYOFFICE Document Editor okruženju i otpremiti ih kao WordPress članak.

Počevši od ONLYOFFICE Docs 8.2, nijedan plugin ne dolazi sa editorima po defaultu. Pluginovi se instaliraju putem Plugin Manager-a.

Povezivanje na WordPress

  1. Otvorite svoj dokument.
  2. Prebacite se na Plugins tab i izaberite
    WordPress.
  3. Prijavite se na svoj WordPress nalog i izaberite stranicu sajta na koju želite da postavite svoj dokument.
  4. Unesite naslov za svoj članak.
  5. Kliknite Publish da objavite odmah ili Save as draft da sačuvate kao skicu i objavite kasnije sa svog WordPress sajta ili aplikacije.
