Uredi HTML

Ako pišete veb stranicu u tekst editoru i želite da je dobijete kao HTML kod, koristite HTML plugin.

Počev od ONLYOFFICE Docs 8.2, nijedan plugin nije podrazumevano uključen u editore. Pluginovi se instaliraju preko Plugin Managera.

  1. Otvorite karticu Pluginovi i kliknite Get and paste html.
  2. Izaberite potreban sadržaj.
  3. HTML kod izabranog pasusa će biti prikazan u polju plugina na levom panelu. Možete izmeniti kod da biste promenili karakteristike teksta, npr. veličinu fonta ili vrstu fonta, itd.
  4. Kliknite Paste into the document da ubacite tekst sa izmenjenim HTML kodom na trenutnu poziciju kursora u dokumentu.

Takođe možete napisati svoj HTML kod (bez selektovanog sadržaja u dokumentu) i zatim ga ubaciti u dokument.

Za više informacija o HTML pluginu i njegovoj instalaciji, pogledajte stranicu plugina na AppDirectory.
