TRUNC funkcija

Funkcija TRUNC je jedna od matematičkih i trigonometrijskih funkcija. Koristi se za vraćanje broja skraćenog na određeni broj cifara.

Sintaksa

TRUNC(number, [num_digits])

Funkcija TRUNC ima sledeće argumente:

Argument Opis
number Broj koji se skraćuje.
num_digits Broj decimalnih mesta koja se prikazuju. Ovo je opcioni argument. Ako je izostavljen, funkcija će pretpostaviti vrednost 0.

Napomene

Kako primeniti funkciju TRUNC.

Primeri

Slika ispod prikazuje rezultat koji vraća funkcija TRUNC.
