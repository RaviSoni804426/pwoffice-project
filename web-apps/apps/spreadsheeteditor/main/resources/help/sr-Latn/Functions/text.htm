TEXT Funkcija

TEXT funkcija je jedna od tekstualnih i podataka funkcija. Koristi se za konvertovanje vrednosti u tekst u zadatom formatu.

Sintaksa

TEXT(vrednost, format_teksta)

TEXT funkcija ima sledeće argumente:

Argument Opis
value Vrednost koju želite konvertovati u tekst.
format_text Format u kojem će biti prikazani rezultati.

Napomene

Kako primeniti TEXT funkciju.

Primeri

Slika ispod prikazuje rezultat koji vraća TEXT funkcija.
