Napredna podešavanja Prikazivača dijagrama

Diagram Viewer omogućava vam da promenite njegova napredna podešavanja. Da biste im pristupili, otvorite karticu File na gornjoj traci sa alatima i izaberite Advanced Settings opciju.

Napredna podešavanja su grupisana na sledeći način:

Izgled

  1. Interface theme opcija koristi se za promenu šeme boja interfejsa uređivača.
    • Same as system opcija omogućava da uređivač prati temu interfejsa vašeg sistema.
    • Light šema boja obuhvata standardne boje crvene cigle, bele i svetlosive sa manjim kontrastom u UI elementima, pogodnim za rad tokom dana.
    • Classic Light šema boja obuhvata standardne boje crvene cigle, bele i svetlosive.
    • Dark šema boja obuhvata crnu, tamnosivu i svetlosivu boju, pogodnu za rad noću.
    • Contrast Dark šema boja obuhvata crnu, tamnosivu i belu boju sa većim kontrastom u UI elementima i naglašava radnu površinu fajla.
    • Gray šema boja obuhvata svetliju sivu boju i deluje kao ujednačeno svetla šema boja.
  2. Tab style opcija koristi se za promenu načina isticanja kartica: Fill za popunjavanje izabrane kartice tamnijom bojom, ili Line za podvlačenje izabrane kartice.
  3. Koristi boju alatne trake kao pozadinu kartica koristi se za popunjavanje kartica istom bojom kao pozadina interfejsa kartica.

Workspace

  1. Uključite podršku za čitač ekrana opcija koristi se za omogućavanje podrške softverima za čitanje ekrana.
  2. Koristite taster Alt za navigaciju kroz korisnički interfejs pomoću tastature opcija koristi se da omogući korišćenje tastera Alt / Option u prečicama na tastaturi.
  3. Podrazumevan Zoom Value opcija koristi se za podešavanje podrazumevanog nivoa zumiranja, birajući ga iz liste dostupnih opcija od 50% do 500%. Takođe možete izabrati opcije Fit to Page, Fit to Width, ili Last Used.

    Poslednje korišćeno opcija odnosi se na poslednje postavljeno skaliranje tokom trenutne sesije.

  4. Font Hinting opcija koristi se za izbor načina prikaza fontova u Prikazivaču dijagrama.
    • Izaberite As Windows ako vam se dopada način na koji se fontovi obično prikazuju u Windows-u, tj. korišćenjem Windows font hintinga.
    • Izaberite As OS X ako vam se dopada način na koji se fontovi obično prikazuju na Mac-u, tj. bez font hintinga.
    • Izaberite Native ako želite da se tekst prikazuje sa hintingom koji je ugrađen u font fajlove.
    • Podrazumevani cache mode - koristi se za izbor režima keširanja karaktera fonta. Ne preporučuje se menjanje bez potrebe. Može biti korisno samo u nekim slučajevima, na primer, kada dođe do problema u Google Chrome pregledaču sa uključenim hardverskim ubrzanjem.

      Prikazivač dijagrama ima dva cache moda:

      1. U prvom cache modu, svako slovo se kešira kao posebna slika.
      2. U Drugom cache modu, bira se slika određene veličine u koju se slova dinamički smeštaju, a implementiran je i mehanizam za dodelu/uklanjanje memorije u ovoj slici. Ako nema dovoljno memorije, kreira se druga slika itd.

      Podrazumevani cache mode podešavanje primenjuje ova dva režima keširanja različito u zavisnosti od pregledača:

      • Kada je Podrazumevani cache mode uključen, Internet Explorer (v. 9, 10, 11) koristi drugi cache mode, a ostali pregledači prvi cache mode.
      • Kada je Podrazumevani cache mode isključen, Internet Explorer (v. 9, 10, 11) koristi prvi cache mode, a ostali pregledači drugi cache mode.

Da biste sačuvali izvršene promene, kliknite Apply dugme.
