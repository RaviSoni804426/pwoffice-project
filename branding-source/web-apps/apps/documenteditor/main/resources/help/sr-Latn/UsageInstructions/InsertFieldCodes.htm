Umetni kodove polja

U Document Editor-u možete dodati polja sa dinamičkim sadržajem koji zavisi od informacija o sistemu ili drugog sadržaja. Nemojte mešati polja sa kontrolama sadržaja, jer su to dve različite funkcionalnosti.

Da biste umetnuli polje,

  1. Idite na karticu Umetanje.
  2. Kliknite na dugme Polje.
  3. U otvorenom dijaloškom prozoru unesite kod polja koji želite da se prikaže i kliknite OK.

Kodovi polja su sledeći:

Kod polja u ONLYOFFICE-u Opis Sintaksa Prekidači
MERGEFIELD Kod umeće vezu ka polju podataka iz drugog izvora u trenutni dokument. Prilikom spajanja, podaci iz drugog izvora zameniće trenutnu vrednost polja. { MERGEFIELD FieldName [ Switches ]}
  1. \b - određuje tekst koji će se umetnuti pre polja MergeField, ako ono nije prazno.
  2. \f - određuje tekst koji će se umetnuti posle polja MergeField, ako ono nije prazno.
  3. \m - određuje da je polje MergeField mapirano.
  4. \v - omogućava konverziju znakova za vertikalno formatiranje.
PAGENUM Kod umeće broj trenutne stranice.
PAGECOUNT Kod umeće ukupan broj stranica u trenutnom dokumentu.
FORMTEXT
TOC Kod kreira sadržaj dokumenta. Sadržaj se zasniva na primenjenim stilovima naslova, nivoima strukture, natpisima itd., koji su definisani poljima za unos u sadržaj (Table of Contents Entry). { TOC [Switches ] }
  1. \a - kreira listu objekata koja ne uključuje oznake i brojeve.
  2. \b - kreira sadržaj za deo dokumenta koji je obeležen oznakom (bookmark).
  3. \с - kreira listu objekata određenog tipa.
  4. \d - određuje tip razdvajača za brojeve stranica.
  5. \f - kreira sadržaj na osnovu TC polja, a ne nivoa strukture dokumenta.
  6. \h - primenjuje formatiranje hiperveze na sadržaj.
  7. \l - određuje potrebne nivoe TC polja. Nivoi koji nisu uključeni u navedeni opseg biće izostavljeni.
  8. \n - kreira sadržaj u potpunosti ili koristeći samo elemente iz navedenog opsega (npr. „1-9“), bez prikaza brojeva stranica.
  9. \o - kreira sadržaj koristeći samo strukturne nivoe dokumenta, a ne TC polja.
  10. \p - određuje razdvajač između elementa i odgovarajućeg broja stranice.
  11. \s - dodaje redni broj broju stranice.
  12. \t - kreira sadržaj koristeći samo prilagođene stilove umesto podrazumevanih stilova.
  13. \u - kreira sadržaj koristeći samo primenjeni nivo strukture formatiranja pasusa.
  14. \w - zadržava tabulacije u unosima sadržaja.
  15. \x - zadržava prelome redova u unosima sadržaja.
  16. \z - skriva brojeve stranica prilikom prikaza sadržaja u rasporedu veb dokumenta.
PAGEREF Kod umeće broj stranice kao oznaku za unakrsnu referencu. { PAGEREF Bookmark [\* Format Switch ] }
  1. \h - kreira hipervezu ka navedenom pasusu.
  2. \p - umeće poziciju navedenog pasusa u odnosu na oznaku.
PAGE Kod umeće broj trenutne stranice. { PAGE [\* Format Switch ] }
NUMPAGES Kod umeće ukupan broj stranica u dokumentu. { NUMPAGES }
ASK Kod umeće polje koje traži od korisnika da unese podatke koji se zatim čuvaju kao oznaka (bookmark). Uređivač će tražiti unos svaki put kada se ovo polje osveži ili tokom spajanja dokumenata. { ASK Bookmark "Prompt" [Optional switches ] }
  1. \d - umeće podrazumevani zamenski tekst kao oznaku.
  2. \o - traži unos samo jednom tokom spajanja dokumenata.
REF Kod umeće sadržaj oznake (bookmark). Oznaka mora biti prethodno umetnuta u dokument. { [REF] Bookmark [Switches ] }
  1. \d - određuje razdvajač za broj stranice.
  2. \f - uvećava brojeve fusnota ili napomena i umeće ih u dokument.
  3. \h - kreira hipervezu ka navedenom pasusu.
  4. \n - dodaje brojeve pasusa, ako postoje.
  5. \p - umeće poziciju navedenog pasusa u odnosu na oznaku.
  6. \r - umeće relativni broj navedenog pasusa.
  7. \t - izostavlja simbole koji nisu numerički ili razdvajači.
  8. \w - umeće puni broj navedenog pasusa.
HYPERLINK Kod umeće hipervezu ka elementu u dokumentu, dokumentu na vašem hard disku, portalu ili resursu treće strane. { HYPERLINK "Filename" [Switches ] }
  1. \l - određuje element unutar dokumenta na koji se prelazi.
  2. \m - određuje koordinate hiperveze za mapu slike na serverskoj strani.
  3. \n - otvara odredišni dokument u novom prozoru.
  4. \o - određuje tekst oblačića (ScreenTip).
  5. \t - određuje cilj hiperveze.
TIME Kod umeće trenutno vreme u dokument na osnovu korisničkih sistemskih podešavanja. { TIME [\@ "Date-Time Picture"] }
DATE Kod umeće trenutni datum u dokument na osnovu korisničkih sistemskih podešavanja. { DATE [ \@ "Date-Time Picture"] [Switches] }
FORMULA Kod izvršava izračunavanje prema unetoj formuli. { = Formula [Bookmark ] [\# Numeric Picture ] }
SEQ Kod omogućava numerisanje poglavlja, tabela, slika i drugih elemenata u dokumentu redom. Ako dodate, uklonite ili premestite stavku zajedno sa njenim SEQ poljem, možete ažurirati ostala SEQ polja u dokumentu kako bi prikazala izmenjeni redosled. { SEQ Identifier [Bookmark ] [Switches ] }
  1. \c - umeće najbliži prethodni redni broj sekvence.
  2. \h - skriva rezultat polja (koristi se za unakrsne reference).
  3. \n - umeće sledeći redni broj sekvence.
  4. \r - numeracija počinje navedenim brojem.
  5. \s - numeracija počinje od navedenog nivoa strukture.
STYLEREF Kod dodaje tekst koji je formatiran pomoću određenog stila. Kada se koristi u zaglavlju ili podnožju, prikazuje prvu ili poslednju pojavu teksta formatiranog tim stilom iz glavnog sadržaja trenutne stranice, omogućavajući kreiranje zaglavlja ili podnožja u stilu rečnika. { STYLEREF StyleIdentifier [Switches ] }
  1. \l - započinje pretragu od dna trenutne stranice.
  2. \n - umeće numeraciju iz navedenog pasusa.
  3. \p - umeće poziciju navedenog pasusa u odnosu na oznaku.
  4. \r - umeće relativni broj navedenog pasusa.
  5. \t - izostavlja simbole koji nisu numerički ili razdvajači.
  6. \w - umeće puni broj navedenog pasusa.
NOTEREF Kod umeće oznaku reference fusnote ili krajnje napomene koja je obeležena oznakom (bookmark), kako bi se omogućile reference na istu napomenu ili unakrsne reference između fusnota i krajnjih napomena. Ako se redosled napomena promeni, rezultat polja će odražavati novu numeraciju. { NOTEREF Bookmark [Switches] }
  1. \f - umeće oznaku reference formatiranu stilom Footnote reference ili Endnote reference.
  2. \h - kreira hipervezu ka navedenoj fusnoti ili krajnjoj napomeni.
  3. \p - umeće relativnu poziciju fusnote ili krajnje napomene.

Kodovi polja mogu se uređivati i ažurirati u hodu:

  1. Izaberite željeno polje i kliknite desnim tasterom miša da biste otvorili kontekstni meni.
    • Ažuriraj polje - ažurira informacije u polju.
    • Uredi polje - otvara prozor za uređivanje polja.
    • Prikaži/sakrij kodove polja - prikazuje kod polja.
