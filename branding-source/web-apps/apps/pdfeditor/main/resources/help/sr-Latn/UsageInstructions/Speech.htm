Pročitajte tekst naglas

ONLYOFFICE PDF Editor ima plugin koji može da pročita tekst umesto vas.

Počevši od ONLYOFFICE Docs verzije 8.2, nijedan plugin ne dolazi podrazumevano uz editore. Plugin-ovi se instaliraju putem Plugin Manager.

  1. Izaberite tekst koji želite da bude pročitan.
  2. Prebacite se na Plugins karticu i izaberite
    Speech.

Tekst će zatim biti pročitan naglas.
