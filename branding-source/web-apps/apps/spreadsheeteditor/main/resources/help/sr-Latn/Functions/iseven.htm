ISEVEN funkcija

ISEVEN funkcija je jedna od informacionih funkcija. Koristi se za proveru parne vrednosti. Ako ćelija sadrži parnu vrednost, funkcija vraća TRUE. Ako je vrednost neparna, funkcija vraća FALSE.

Sintaksa

ISEVEN(number)

ISEVEN funkcija ima sledeći argument:

Argument Opis
number Numerička vrednost koju želite da testirate.

Napomene

Ako je number nenumerička vrednost, ISEVEN vraća grešku #VALUE!.

Ako number nije ceo broj, vrednost se skraćuje (trunkira).

Kako primeniti ISEVEN funkciju.

Primeri

Slika ispod prikazuje rezultat koji vraća ISEVEN funkcija.
