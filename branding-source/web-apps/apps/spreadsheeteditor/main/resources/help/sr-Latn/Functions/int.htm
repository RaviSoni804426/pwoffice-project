INT funkcija

INT funkcija je jedna od matematičkih i trigonometrijskih funkcija. Koristi se za analizu i vraćanje celobrojnog dela navedenog broja.

Sintaksa

INT(broj)

INT funkcija ima sledeći argument:

Argument Opis
broj Numerička vrednost za koju želite da dobijete celobrojni deo.

Napomene

Ako je vrednost broj negativna, funkcija vraća prvi negativan broj koji je manji ili jednak izabranom.

Kako primeniti INT funkciju.

Primeri

Slika ispod prikazuje rezultat koji vraća INT funkcija.
