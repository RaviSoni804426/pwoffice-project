EFFECT funkcija

EFFECT funkcija je jedna od finansijskih funkcija. Koristi se za izračunavanje efektivne godišnje kamatne stope za vrednosni papir na osnovu zadate nominalne godišnje kamatne stope i broja perioda kapitalizacije godišnje.

Sintaksa

EFFECT(nominal_rate, npery)

EFFECT funkcija ima sledeće argumente:

Argument Opis
nominal_rate Nominalna godišnja kamatna stopa vrednosnog papira.
npery Broj perioda kapitalizacije godišnje.

Napomene

Kako primeniti EFFECT funkciju.

Primeri

Slika ispod prikazuje rezultat koji vraća EFFECT funkcija.
