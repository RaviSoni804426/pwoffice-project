MODE funkcija

MODE funkcija je jedna od statističkih funkcija. Koristi se za analizu opsega podataka i vraćanje najčešće pojavljivane vrednosti.

Sintaksa

MODE(number1, [number2], ...)

MODE funkcija ima sledeće argumente:

Argument Opis
number1/2/n Do 255 numeričkih vrednosti za koje želite da pronađete najčešće pojavljivanu vrednost.

Napomene

Ako nema ponavljajuće vrednosti u listi argumenata, funkcija vraća #VALUE! grešku.

Kako primeniti MODE funkciju.

Primeri

Slika ispod prikazuje rezultat koji vraća MODE funkcija.
